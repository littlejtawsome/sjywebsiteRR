Where To Start

The Basics

  • HTML

    We set up our pages using HTML5, using the most current doctype, and set up to be built responsively. Download our template

  • Interactivity

    The interactivity of our components are built with javascript and jQuery. jQuery can be applied with the latest CDN. Download our template

  • Fonts and Icons

    We have a selection of fonts and icons that we recommend for our pages, but you can choose to use as many or as little as your prefer.

    We like to use GoogleFonts because they have a large selection, you wil be able to find almost any style to suit your needs. We like to use FontAwesome for our icons. Like GoogleFonts, they have a huge selection and are easy to use. Both GoogleFonts and FontAwesome are available through CDN. Download our template
