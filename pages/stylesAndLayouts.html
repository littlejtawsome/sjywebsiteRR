Icons

Using icon fonts is a great way to add not only style but great visual indicators on your website. We like to use FontAwesome as our resource. Simply add the CDN to the <head></head> section of your webpage and add the icon font tags ><i> to your elements. Here are are few that we recommend.

                        
    <head>
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/
        font-awesome/5.13.0/css/all.min.css">
    </head>
